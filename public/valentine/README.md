# Valentine page assets

Add your own files here for the `/will-you-be-my-valentine` page:

- **Slideshow (13 portrait images):**  
  Name them `slide-1.jpeg` through `slide-13.jpeg` (or .jpg / .png).  
  Portrait orientation works best (e.g. 400Ã—600 or similar).

- **Response video (after "Yes!"):**  
  Name it `response-video.mp4`.  
  It will autoplay with sound after the heart explosion.

If these files are missing, the slideshow will show placeholders and the video area will show the player (you can upload the video later).
